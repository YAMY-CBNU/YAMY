# YAMY

간단한 요리 레시피 추천 및 요리 가이드를 보여주는 정적 웹페이지 프로젝트입니다.

`index.html`은 메인 페이지이고, `ramen.html`은 라면 레시피 상세 페이지입니다. 최근 리팩토링으로 HTML 안에 있던 CSS와 JavaScript를 `assets/` 폴더로 분리해 관리하고 있습니다.

## 구성

```text
YAMY/
├─ index.html
├─ ramen.html
├─ asd.html
├─ assets/
│  ├─ css/
│  │  ├─ common.css
│  │  ├─ index.css
│  │  └─ recipe-detail.css
│  └─ js/
│     ├─ index.js
│     └─ ramen.js
└─ README.md
```

## 파일 설명

- `index.html`
  메인 페이지입니다. 레시피 카드 목록, 검색창, 카테고리/난이도 필터 UI를 포함합니다.

- `ramen.html`
  라면 상세 레시피 페이지입니다. 재료 체크리스트, 조리 단계 안내, 타이머, 단계 이동 UI를 포함합니다.

- `assets/css/common.css`
  두 페이지에서 공통으로 쓰는 레이아웃과 기본 스타일입니다. 헤더, 푸터, 공통 컨테이너, 공통 섹션 스타일이 들어 있습니다.

- `assets/css/index.css`
  메인 페이지 전용 스타일입니다. 검색창, 카테고리 버튼, 레시피 카드 목록 등의 스타일을 담당합니다.

- `assets/css/recipe-detail.css`
  상세 페이지 전용 스타일입니다. 재료 체크리스트, 단계별 가이드, 타이머, 알림 UI 스타일을 담당합니다.

- `assets/js/index.js`
  메인 페이지 동작을 담당합니다. 검색, 카테고리 필터, 난이도 필터, 카드 클릭, 카드 애니메이션이 들어 있습니다.

- `assets/js/ramen.js`
  상세 페이지 동작을 담당합니다. 조리 단계 전환, 타이머, 재료 체크 상태 반영, 키보드 이동, 단계 인디케이터 클릭 처리가 들어 있습니다.

- `asd.html`
  예전 구조가 남아 있는 실험용 또는 백업용 파일로 보이며, 현재 메인 흐름에서는 사용하지 않습니다.

## 동작 방식

### 메인 페이지

- 검색창에서 텍스트를 입력하면 레시피 카드 제목과 설명을 기준으로 필터링합니다.
- 카테고리 버튼과 난이도 링크를 눌러 카드 목록을 걸러볼 수 있습니다.
- 라면 카드를 누르면 상세 페이지인 `ramen.html`로 이동합니다.

### 상세 페이지

- 재료 체크박스를 선택하면 아직 준비되지 않은 재료 목록이 갱신됩니다.
- 조리 단계를 이전/다음 버튼, 키보드 좌우 화살표, 인디케이터 점 클릭으로 이동할 수 있습니다.
- 타이머는 단계별 시간에 맞춰 표시되며, 시작/일시정지/초기화가 가능합니다.

## 코드 구조 요약

- HTML은 페이지 구조와 콘텐츠만 담당합니다.
- CSS는 공통 스타일과 페이지별 스타일로 나누어 관리합니다.
- JavaScript는 페이지별로 분리해 각 페이지의 이벤트와 상태를 처리합니다.
- 인라인 `onclick` 대신 `id`, `class`, `data-*` 속성과 `addEventListener()`를 사용합니다.

## 실행 방법

별도 빌드 도구나 패키지 설치가 필요 없는 정적 프로젝트입니다.

1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.
2. 메인 페이지에서 레시피를 확인합니다.
3. 라면 카드 클릭 시 `ramen.html` 상세 페이지로 이동합니다.

## 참고

- 현재 구조는 페이지 수가 많지 않을 때 관리하기 쉬운 형태로 정리되어 있습니다.
- 상세 레시피 페이지가 더 늘어나면 공통 컴포넌트나 레시피 데이터 분리까지 확장할 수 있습니다.
